## 개념 요약
TCP(Transmission Control Protocol)는 연결 지향형 프로토콜로,
데이터를 신뢰성 있게 순서대로 전달.
이 단계의 핵심은 "**서버와 클라이언트 간의 연결이 어떻게 형성되고 유지되는가**"를 직접 구현해보는 것
